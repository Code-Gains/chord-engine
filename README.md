# Disclaimer

Right now I am fixing the architecture after rewriting the engine from DX11 to Vulkan, so there is no propper setup guide for local development. I will update the readme once that is available.

# Purpose
Chord Engine is an experimental Vulkan renderer using the [C++ EnTT Library](https://github.com/skypjack/entt) for managing CPU side data. This project is for self improvement and to satisfy my curiosity about graphics programming.
